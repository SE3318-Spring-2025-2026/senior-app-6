# Sequence Diagram — P2 Sub-Process 2.3
## Process Invitation Response

> Endpoints: `GET /api/invitations/pending`, `PATCH /api/invitations/{invitationId}/respond`
> Issues: B-09, B-15
> JWT principal = internal student UUID
> ⚠️ The ACCEPT path is the most complex transaction in P2 — auto-deny must happen in the same @Transactional call.
> **Implementation note:** Invitation logic lives in `InvitationService` (extracted from `GroupService`).

---

### GET /api/invitations/pending

```mermaid
sequenceDiagram
    actor Student
    participant InvitationController
    participant InvitationService
    participant GroupInvitationRepository

    Student->>InvitationController: GET /api/invitations/pending<br/>Authorization: Bearer <student_jwt>

    InvitationController->>InvitationController: extract studentUUID from SecurityContext

    InvitationController->>InvitationService: getPendingInvitations(studentUUID)

    InvitationService->>GroupInvitationRepository: findByInviteeIdAndStatus(studentUUID, PENDING)
    GroupInvitationRepository-->>InvitationService: List<GroupInvitation>

    Note over InvitationService: Empty list is valid — never 404
    InvitationService-->>InvitationController: List<InvitationResponse>
    InvitationController-->>Student: 200 [{invitationId, groupId, groupName, teamLeaderStudentId, sentAt}, ...]
```

### PATCH /api/invitations/{invitationId}/respond — DECLINED path

```mermaid
sequenceDiagram
    actor Student
    participant InvitationController
    participant InvitationService
    participant GroupInvitationRepository
    participant AuditLogService

    Student->>InvitationController: PATCH /api/invitations/{invitationId}/respond<br/>{accept: false}<br/>Authorization: Bearer <student_jwt>

    InvitationController->>InvitationController: extract studentUUID from SecurityContext
    InvitationController->>InvitationService: respondToInvitation(invitationId, studentUUID, accept=false)

    InvitationService->>GroupInvitationRepository: findById(invitationId)
    GroupInvitationRepository-->>InvitationService: Optional<GroupInvitation>

    alt not found
        InvitationService-->>InvitationController: throw InvitationNotFoundException
        InvitationController-->>Student: 404 {"error": "Invitation not found"}
    end

    alt invitation.inviteeId != studentUUID
        InvitationService-->>InvitationController: throw ForbiddenException
        InvitationController-->>Student: 403 {"error": "This invitation does not belong to you"}
    end

    alt invitation.status != PENDING
        InvitationService-->>InvitationController: throw InvitationNotPendingException
        InvitationController-->>Student: 400 {"error": "Invitation is no longer pending"}
    end

    InvitationService->>GroupInvitationRepository: save(invitation{status=DECLINED, respondedAt=now})
    GroupInvitationRepository-->>InvitationService: ok

    InvitationService->>AuditLogService: record(studentUUID, STUDENT, INVITATION_RESPONDED, GROUP, SUCCESS)

    InvitationService-->>InvitationController: {invitationId, status:"DECLINED", respondedAt}
    InvitationController-->>Student: 200 {invitationId, status:"DECLINED", respondedAt}
```

### PATCH /api/invitations/{invitationId}/respond — ACCEPTED path

```mermaid
sequenceDiagram
    actor Student
    participant InvitationController
    participant InvitationService
    participant GroupInvitationRepository
    participant ProjectGroupRepository
    participant GroupMembershipRepository
    participant GroupService
    participant AuditLogService

    Student->>InvitationController: PATCH /api/invitations/{invitationId}/respond<br/>{accept: true}<br/>Authorization: Bearer <student_jwt>

    InvitationController->>InvitationController: extract studentUUID from SecurityContext
    InvitationController->>InvitationService: respondToInvitation(invitationId, studentUUID, accept=true)

    InvitationService->>GroupInvitationRepository: findById(invitationId)
    GroupInvitationRepository-->>InvitationService: Optional<GroupInvitation>

    alt not found
        InvitationService-->>InvitationController: throw InvitationNotFoundException
        InvitationController-->>Student: 404 {"error": "Invitation not found"}
    end

    alt invitation.inviteeId != studentUUID
        InvitationService-->>InvitationController: throw ForbiddenException
        InvitationController-->>Student: 403 {"error": "This invitation does not belong to you"}
    end

    alt invitation.status != PENDING
        InvitationService-->>InvitationController: throw InvitationNotPendingException
        InvitationController-->>Student: 400 {"error": "Invitation is no longer pending"}
    end

    Note over InvitationService: Roster lock — fetch group and check status via locksRoster()
    InvitationService->>ProjectGroupRepository: findById(invitation.groupId)
    ProjectGroupRepository-->>InvitationService: ProjectGroup

    alt group.status == DISBANDED
        InvitationService-->>InvitationController: throw BusinessRuleException
        InvitationController-->>Student: 400 {"error": "This group has been disbanded"}
    end

    alt group.status.locksRoster() (TOOLS_BOUND or ADVISOR_ASSIGNED)
        InvitationService-->>InvitationController: throw BusinessRuleException
        InvitationController-->>Student: 400 {"error": "Group roster is locked after tool binding"}
    end

    Note over InvitationService: Max team size check — only current members (accept consumes one pending seat)
    InvitationService->>GroupMembershipRepository: countByGroupId(invitation.groupId)
    GroupMembershipRepository-->>InvitationService: memberCount

    alt memberCount >= maxTeamSize
        InvitationService-->>InvitationController: throw BusinessRuleException
        InvitationController-->>Student: 400 {"error": "Group has reached maximum team size"}
    end

    Note over InvitationService: Guard: student may have joined another group since invitation was sent
    InvitationService->>GroupMembershipRepository: existsByStudentId(studentUUID)
    GroupMembershipRepository-->>InvitationService: boolean

    alt student already in a group
        InvitationService-->>InvitationController: throw BusinessRuleException
        InvitationController-->>Student: 400 {"error": "You are already a member of a group"}
    end

    Note over InvitationService: @Transactional — all writes are atomic.<br/>If any step fails, nothing is persisted.

    InvitationService->>GroupInvitationRepository: saveAndFlush(invitation{status=ACCEPTED, respondedAt=now})
    GroupInvitationRepository-->>InvitationService: ok

    InvitationService->>GroupMembershipRepository: save(GroupMembership{studentId, groupId, role=MEMBER, joinedAt=now})
    GroupMembershipRepository-->>InvitationService: ok

    Note over InvitationService: Bulk-update ALL other PENDING invitations<br/>for this student across ALL groups → AUTO_DENIED<br/>Excludes the accepted invitation ID
    InvitationService->>GroupInvitationRepository: autoDenyOtherPendingInvitationsExcept(studentUUID, invitationId)
    GroupInvitationRepository-->>InvitationService: updatedCount

    InvitationService->>AuditLogService: record(studentUUID, STUDENT, INVITATION_RESPONDED, GROUP, SUCCESS)

    Note over InvitationService: Delegates to GroupService for fresh group detail
    InvitationService->>GroupService: getGroupDetail(group.id)
    GroupService-->>InvitationService: GroupDetailResponse

    InvitationService-->>InvitationController: GroupDetailResponse (full group detail)
    InvitationController-->>Student: 200 {id, groupName, termId, status, members:[...]}
```
