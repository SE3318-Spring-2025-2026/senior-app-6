# Sequence Diagram — P2 Sub-Process 2.2
## Search Student & Send / Cancel Invitation

> Endpoints: `GET /api/students/search`, `POST /api/groups/{groupId}/invitations`, `GET /api/groups/{groupId}/invitations`, `DELETE /api/invitations/{invitationId}`
> Issues: B-04, B-05, B-09, B-15, B-17
> JWT principal = internal student UUID
> **Implementation note:** Invitation logic lives in `InvitationService` (extracted from `GroupService`).

### GET /api/students/search?q={query}

```mermaid
sequenceDiagram
    actor Student
    participant StudentSearchController
    participant StudentRepository

    Student->>StudentSearchController: GET /api/students/search?q=230<br/>Authorization: Bearer <student_jwt>

    alt q.length < 3
        StudentSearchController-->>Student: 400 {"error": "Query must be at least 3 characters"}
    end

    Note over StudentSearchController: Query students not in any group.<br/>Join student LEFT JOIN group_membership WHERE membership IS NULL<br/>+ studentId LIKE '%q%'
    StudentSearchController->>StudentRepository: findUngroupedByStudentIdContaining(q)
    StudentRepository-->>StudentSearchController: List<Student>

    StudentSearchController-->>Student: 200 [{studentId, githubUsername}]
```

### POST /api/groups/{groupId}/invitations

```mermaid
sequenceDiagram
    actor Student as Student (Team Leader)
    participant GroupController
    participant InvitationService
    participant ProjectGroupRepository
    participant GroupMembershipRepository
    participant StudentRepository
    participant GroupInvitationRepository
    participant AuditLogService

    Student->>GroupController: POST /api/groups/{groupId}/invitations<br/>{targetStudentId: "23070006036"}<br/>Authorization: Bearer <student_jwt>

    GroupController->>GroupController: extract requesterUUID from SecurityContext

    GroupController->>InvitationService: sendInvitation(groupId, requesterUUID, targetStudentId)

    InvitationService->>ProjectGroupRepository: findById(groupId)
    ProjectGroupRepository-->>InvitationService: Optional<ProjectGroup>

    alt group not found
        InvitationService-->>GroupController: throw GroupNotFoundException
        GroupController-->>Student: 404 {"error": "Group not found"}
    end

    InvitationService->>InvitationService: requireTeamLeader(groupId, requesterUUID)

    alt requester is not TEAM_LEADER of this group
        InvitationService-->>GroupController: throw ForbiddenException
        GroupController-->>Student: 403 {"error": "Only the Team Leader can send invitations"}
    end

    alt group.status == DISBANDED
        InvitationService-->>GroupController: throw BusinessRuleException
        GroupController-->>Student: 400 {"error": "Cannot send invitation from a disbanded group"}
    end

    Note over InvitationService: Max team size check — members + pending outbound invitations
    InvitationService->>GroupMembershipRepository: countByGroupId(groupId)
    InvitationService->>GroupInvitationRepository: countByGroupIdAndStatus(groupId, PENDING)
    GroupMembershipRepository-->>InvitationService: memberCount
    GroupInvitationRepository-->>InvitationService: pendingCount

    alt memberCount + pendingCount >= maxTeamSize
        InvitationService-->>GroupController: throw BusinessRuleException
        GroupController-->>Student: 400 {"error": "Group has reached maximum team size"}
    end

    InvitationService->>StudentRepository: findByStudentId(targetStudentId)
    StudentRepository-->>InvitationService: Optional<Student>

    alt student not found
        InvitationService-->>GroupController: throw BusinessRuleException
        GroupController-->>Student: 400 {"error": "Student '{id}' is not registered in the system"}
    end

    InvitationService->>GroupMembershipRepository: existsByStudentId(targetStudent.id)
    GroupMembershipRepository-->>InvitationService: boolean

    alt target already in a group
        InvitationService-->>GroupController: throw AlreadyInGroupException
        GroupController-->>Student: 400 {"error": "Student '{id}' is already a member of a group"}
    end

    InvitationService->>GroupInvitationRepository: existsByGroupIdAndInviteeIdAndStatus(groupId, targetId, PENDING)
    GroupInvitationRepository-->>InvitationService: boolean

    alt pending invitation already exists
        InvitationService-->>GroupController: throw DuplicateInvitationException
        GroupController-->>Student: 409 {"error": "A pending invitation already exists for this student"}
    end

    InvitationService->>GroupInvitationRepository: save(GroupInvitation{status=PENDING})
    GroupInvitationRepository-->>InvitationService: savedInvitation

    InvitationService->>AuditLogService: record(requesterUUID, STUDENT, INVITATION_SENT, GROUP, SUCCESS)

    InvitationService-->>GroupController: InvitationResponse
    GroupController-->>Student: 201 {invitationId, groupId, targetStudentId, status:"PENDING", sentAt}
```

### GET /api/groups/{groupId}/invitations

```mermaid
sequenceDiagram
    actor Student as Student (Team Leader)
    participant GroupController
    participant InvitationService
    participant ProjectGroupRepository
    participant GroupMembershipRepository
    participant GroupInvitationRepository

    Student->>GroupController: GET /api/groups/{groupId}/invitations<br/>Authorization: Bearer <student_jwt>

    GroupController->>GroupController: extract requesterUUID from SecurityContext
    GroupController->>InvitationService: getGroupInvitations(groupId, requesterUUID)

    InvitationService->>ProjectGroupRepository: findById(groupId)
    ProjectGroupRepository-->>InvitationService: Optional<ProjectGroup>

    alt group not found
        InvitationService-->>GroupController: throw GroupNotFoundException
        GroupController-->>Student: 404 {"error": "Group not found"}
    end

    InvitationService->>InvitationService: requireTeamLeader(groupId, requesterUUID)

    alt not TEAM_LEADER

    end

    InvitationService->>GroupInvitationRepository: findByGroupId(groupId)
    GroupInvitationRepository-->>InvitationService: List<GroupInvitation> (all statuses)

    InvitationService-->>GroupController: List<InvitationResponse>
    GroupController-->>Student: 200 [{invitationId, targetStudentId, status, sentAt, respondedAt}, ...]
```

### DELETE /api/invitations/{invitationId}

```mermaid
sequenceDiagram
    actor Student as Student (Team Leader)
    participant InvitationController
    participant InvitationService
    participant GroupInvitationRepository

    Student->>InvitationController: DELETE /api/invitations/{invitationId}<br/>Authorization: Bearer <student_jwt>

    InvitationController->>InvitationController: extract requesterUUID from SecurityContext
    InvitationController->>InvitationService: cancelInvitation(invitationId, requesterUUID)

    InvitationService->>GroupInvitationRepository: findById(invitationId)
    GroupInvitationRepository-->>InvitationService: Optional<GroupInvitation>

    alt not found
        InvitationService-->>InvitationController: throw InvitationNotFoundException
        InvitationController-->>Student: 404 {"error": "Invitation not found"}
    end

    InvitationService->>InvitationService: requireTeamLeader(invitation.group.id, requesterUUID)

    alt requester is not TEAM_LEADER of the inviting group
        InvitationService-->>InvitationController: throw ForbiddenException
        InvitationController-->>Student: 403 {"error": "Only the Team Leader can cancel invitations"}
    end

    alt invitation.status != PENDING
        InvitationService-->>InvitationController: throw InvitationNotPendingException
        InvitationController-->>Student: 400 {"error": "Invitation is no longer pending"}
    end

    InvitationService->>GroupInvitationRepository: save(invitation{status=CANCELLED, respondedAt=now})
    GroupInvitationRepository-->>InvitationService: ok

    InvitationService-->>InvitationController: {invitationId, status:"CANCELLED", respondedAt}
    InvitationController-->>Student: 200 {invitationId, status:"CANCELLED", respondedAt}
```
