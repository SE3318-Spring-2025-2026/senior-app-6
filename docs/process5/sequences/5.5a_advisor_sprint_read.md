### GET /api/advisor/sprints/active

```mermaid
sequenceDiagram
    actor Advisor
    participant AdvisorSprintController
    participant SprintService
    participant TermConfigService
    participant SprintRepository

    Advisor->>AdvisorSprintController: GET /api/advisor/sprints/active
    Note over AdvisorSprintController: Authorization: Bearer <staff_jwt> role=PROFESSOR

    AdvisorSprintController->>SprintService: getActiveSprint()

    Note over SprintService: termId resolved server-side — never from client
    SprintService->>TermConfigService: getActiveTermId()
    TermConfigService-->>SprintService: termId

    SprintService->>SprintRepository: findByTermIdAndStartDateLessThanEqualAndEndDateGreaterThanEqual(termId, today, today)
    SprintRepository-->>SprintService: Optional<Sprint>

    alt no active sprint found
        SprintService-->>AdvisorSprintController: throw NotFoundException
        AdvisorSprintController-->>Advisor: 404 {"message": "No active sprint found for the current term"}
    end

    Note over SprintService: daysRemaining = sprint.endDate - today (computed, not stored)
    SprintService-->>AdvisorSprintController: SprintSummaryResponse
    AdvisorSprintController-->>Advisor: 200 { sprintId, startDate, endDate, storyPointTarget, daysRemaining }
```

### GET /api/advisor/sprints/{sprintId}/groups

```mermaid
sequenceDiagram
    actor Advisor
    participant AdvisorSprintController
    participant ScrumGradingService
    participant SprintRepository
    participant ProjectGroupRepository
    participant SprintTrackingLogRepository
    participant ScrumGradeRepository

    Advisor->>AdvisorSprintController: GET /api/advisor/sprints/{sprintId}/groups
    Note over AdvisorSprintController: Authorization: Bearer <staff_jwt> role=PROFESSOR

    AdvisorSprintController->>AdvisorSprintController: extract advisorId from SecurityContext

    AdvisorSprintController->>ScrumGradingService: getAdvisorGroupSummaries(advisorId, sprintId)

    ScrumGradingService->>SprintRepository: findById(sprintId)
    SprintRepository-->>ScrumGradingService: Optional<Sprint>

    alt sprint not found
        ScrumGradingService-->>AdvisorSprintController: throw NotFoundException
        AdvisorSprintController-->>Advisor: 404 {"message": "Sprint not found"}
    end

    Note over ScrumGradingService: termId resolved via TermConfigService.getActiveTermId()
    ScrumGradingService->>ProjectGroupRepository: findByAdvisor_IdAndTermId(advisorId, termId)
    ProjectGroupRepository-->>ScrumGradingService: List<ProjectGroup>

    loop for each group
        ScrumGradingService->>SprintTrackingLogRepository: findByGroupIdAndSprintId(group.id, sprintId)
        SprintTrackingLogRepository-->>ScrumGradingService: List<SprintTrackingLog>

        ScrumGradingService->>ScrumGradeRepository: findByGroupIdAndSprintId(group.id, sprintId)
        ScrumGradeRepository-->>ScrumGradingService: Optional<ScrumGrade>

        Note over ScrumGradingService: aggregate: totalIssues, mergedPRs, aiPassCount, aiWarnCount, aiFailCount, aiSkippedCount, aiPendingCount
        Note over ScrumGradingService: perStudentSummary: GROUP BY assigneeGithubUsername
        Note over ScrumGradingService: aiValidationStatus per student = worst(aiPrResult, aiDiffResult) — FAIL > WARN > SKIPPED > PASS > PENDING
    end

    ScrumGradingService-->>AdvisorSprintController: List<GroupSummaryResponse>
    AdvisorSprintController-->>Advisor: 200 [{ groupId, groupName, totalIssues, mergedPRs, aiCounts, gradeSubmitted, pointA_grade, pointB_grade, perStudentSummary }, ...]
```

### GET /api/advisor/sprints/{sprintId}/groups/{groupId}/tracking

```mermaid
sequenceDiagram
    actor Advisor
    participant AdvisorSprintController
    participant ScrumGradingService
    participant ProjectGroupRepository
    participant SprintRepository
    participant SprintTrackingLogRepository

    Advisor->>AdvisorSprintController: GET /api/advisor/sprints/{sprintId}/groups/{groupId}/tracking
    Note over AdvisorSprintController: Authorization: Bearer <staff_jwt> role=PROFESSOR

    AdvisorSprintController->>AdvisorSprintController: extract advisorId from SecurityContext

    AdvisorSprintController->>ScrumGradingService: getGroupTracking(advisorId, groupId, sprintId)

    ScrumGradingService->>ProjectGroupRepository: findById(groupId)
    ProjectGroupRepository-->>ScrumGradingService: Optional<ProjectGroup>

    alt group not found
        ScrumGradingService-->>AdvisorSprintController: throw GroupNotFoundException
        AdvisorSprintController-->>Advisor: 404 {"message": "Group not found"}
    end

    Note over ScrumGradingService: Ownership check: group.getAdvisor() == null OR group.getAdvisor().getId() != advisorId
    Note over ScrumGradingService: Use group.getAdvisor().getId() — there is no group.getAdvisorId() field

    alt ownership check fails
        ScrumGradingService-->>AdvisorSprintController: throw ForbiddenException
        AdvisorSprintController-->>Advisor: 403 {"message": "You are not the advisor for this group"}
    end

    ScrumGradingService->>SprintRepository: findById(sprintId)
    SprintRepository-->>ScrumGradingService: Optional<Sprint>

    alt sprint not found
        ScrumGradingService-->>AdvisorSprintController: throw NotFoundException
        AdvisorSprintController-->>Advisor: 404 {"message": "Sprint not found"}
    end

    ScrumGradingService->>SprintTrackingLogRepository: findByGroupIdAndSprintId(groupId, sprintId)
    SprintTrackingLogRepository-->>ScrumGradingService: List<SprintTrackingLog>

    alt issues is empty
        ScrumGradingService-->>AdvisorSprintController: throw NotFoundException
        AdvisorSprintController-->>Advisor: 404 {"message": "No tracking data found — sprint may not have been processed yet"}
    end

    Note over ScrumGradingService: perStudentSummary: same aggregation as /groups endpoint
    ScrumGradingService-->>AdvisorSprintController: GroupTrackingResponse
    AdvisorSprintController-->>Advisor: 200 { groupId, sprintId, fetchedAt, issues: [...], perStudentSummary: [...] }
```
