### GET /api/sprints/active

```mermaid
sequenceDiagram
    actor Student
    participant StudentSprintController
    participant SprintService
    participant TermConfigService
    participant SprintRepository

    Student->>StudentSprintController: GET /api/sprints/active
    Note over StudentSprintController: Authorization: Bearer <student_jwt>

    StudentSprintController->>SprintService: getActiveSprint()

    Note over SprintService: termId resolved server-side — never from client
    SprintService->>TermConfigService: getActiveTermId()
    TermConfigService-->>SprintService: termId

    SprintService->>SprintRepository: findByTermIdAndStartDateLessThanEqualAndEndDateGreaterThanEqual(termId, today, today)
    SprintRepository-->>SprintService: Optional<Sprint>

    alt no active sprint found
        SprintService-->>StudentSprintController: throw NotFoundException
        StudentSprintController-->>Student: 404 {"message": "No active sprint found for the current term"}
    end

    Note over SprintService: daysRemaining = sprint.endDate - today (computed, not stored)
    SprintService-->>StudentSprintController: SprintSummaryResponse
    StudentSprintController-->>Student: 200 { sprintId, startDate, endDate, storyPointTarget, daysRemaining }
```

### GET /api/groups/{groupId}/sprints/{sprintId}/tracking

```mermaid
sequenceDiagram
    actor Student
    participant StudentSprintController
    participant SprintService
    participant GroupMembershipRepository
    participant SprintRepository
    participant SprintTrackingLogRepository

    Student->>StudentSprintController: GET /api/groups/{groupId}/sprints/{sprintId}/tracking
    Note over StudentSprintController: Authorization: Bearer <student_jwt>
    Note over StudentSprintController: groupId resolved on frontend from GET /api/groups/my — not from URL manipulation

    StudentSprintController->>StudentSprintController: extract studentId from JWT claim

    StudentSprintController->>SprintService: getGroupTracking(studentId, groupId, sprintId)

    SprintService->>GroupMembershipRepository: existsByGroupIdAndStudentId(groupId, studentId)
    GroupMembershipRepository-->>SprintService: boolean

    alt student is not a member of this group
        SprintService-->>StudentSprintController: throw ForbiddenException
        StudentSprintController-->>Student: 403 {"message": "You are not a member of this group"}
    end

    SprintService->>SprintRepository: findById(sprintId)
    SprintRepository-->>SprintService: Optional<Sprint>

    alt sprint not found
        SprintService-->>StudentSprintController: throw NotFoundException
        StudentSprintController-->>Student: 404 {"message": "Sprint not found"}
    end

    SprintService->>SprintTrackingLogRepository: findByGroupIdAndSprintId(groupId, sprintId)
    SprintTrackingLogRepository-->>SprintService: List<SprintTrackingLog>

    alt issues is empty — pipeline not yet run
        Note over StudentSprintController: Return 200 with empty array — NOT 404
        SprintService-->>StudentSprintController: TrackingResponse (empty)
        StudentSprintController-->>Student: 200 { groupId, sprintId, fetchedAt: null, issues: [] }

    else issues have data
        Note over SprintService: fetchedAt = max(fetchedAt) across all log rows
        SprintService-->>StudentSprintController: TrackingResponse
        StudentSprintController-->>Student: 200 { groupId, sprintId, fetchedAt, issues: [{ issueKey, assigneeGithubUsername, storyPoints, prNumber, prMerged, aiPrResult, aiDiffResult }, ...] }
    end
```
