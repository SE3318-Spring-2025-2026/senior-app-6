### GET /api/sprints/active

```mermaid
sequenceDiagram
    actor Student
    participant SprintController
    participant SprintService
    participant TermConfigService
    participant SprintRepository

    Student->>SprintController: GET /api/sprints/active
    Note over SprintController: Authorization: Bearer <student_jwt>
    Note over SprintController: New controller — @RequestMapping("/api/sprints")

    SprintController->>SprintService: getActiveSprint()

    Note over SprintService: termId resolved server-side — never from client
    SprintService->>TermConfigService: getActiveTermId()
    TermConfigService-->>SprintService: termId

    SprintService->>SprintRepository: findByTermIdAndStartDateLessThanEqualAndEndDateGreaterThanEqual(termId, today, today)
    SprintRepository-->>SprintService: Optional<Sprint>

    alt no active sprint found
        SprintService-->>SprintController: throw NotFoundException
        SprintController-->>Student: 404 {"message": "No active sprint found for the current term"}
    end

    Note over SprintService: daysRemaining = sprint.endDate - today (computed, not stored)
    SprintService-->>SprintController: SprintSummaryResponse
    SprintController-->>Student: 200 { sprintId, startDate, endDate, storyPointTarget, daysRemaining }
```

### GET /api/groups/{groupId}/sprints/{sprintId}/tracking

```mermaid
sequenceDiagram
    actor Student
    participant GroupController
    participant SprintService
    participant GroupMembershipRepository
    participant SprintRepository
    participant SprintTrackingLogRepository

    Student->>GroupController: GET /api/groups/{groupId}/sprints/{sprintId}/tracking
    Note over GroupController: Authorization: Bearer <student_jwt>
    Note over GroupController: groupId resolved on frontend from GET /api/groups/my — not from URL manipulation

    GroupController->>GroupController: extract studentUUID from JWT claim (id)

    GroupController->>SprintService: getGroupTracking(studentUUID, groupId, sprintId)

    SprintService->>GroupMembershipRepository: existsByGroupIdAndStudentId(groupId, studentUUID)
    GroupMembershipRepository-->>SprintService: boolean

    alt student is not a member of this group
        SprintService-->>GroupController: throw ForbiddenException
        GroupController-->>Student: 403 {"message": "You are not a member of this group"}
    end

    SprintService->>SprintRepository: findById(sprintId)
    SprintRepository-->>SprintService: Optional<Sprint>

    alt sprint not found
        SprintService-->>GroupController: throw NotFoundException
        GroupController-->>Student: 404 {"message": "Sprint not found"}
    end

    SprintService->>SprintTrackingLogRepository: findByGroupIdAndSprintId(groupId, sprintId)
    SprintTrackingLogRepository-->>SprintService: List<SprintTrackingLog>

    alt issues is empty — pipeline not yet run
        Note over GroupController: Return 200 with empty array — NOT 404
        SprintService-->>GroupController: TrackingResponse (empty)
        GroupController-->>Student: 200 { groupId, sprintId, fetchedAt: null, issues: [] }

    else issues have data
        Note over SprintService: fetchedAt = max(fetchedAt) across all log rows
        SprintService-->>GroupController: TrackingResponse
        GroupController-->>Student: 200 { groupId, sprintId, fetchedAt, issues: [{ issueKey, assigneeGithubUsername, storyPoints, prNumber, prMerged, aiPrResult, aiDiffResult }, ...] }
    end
```
