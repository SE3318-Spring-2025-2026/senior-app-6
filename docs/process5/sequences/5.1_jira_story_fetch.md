> Spec: IR-2, P5 Step 2, NFR-7 (token encrypted at rest)

---

### JiraSprintService.fetchSprintStories(group)

```mermaid
sequenceDiagram
    participant SprintTrackingOrchestrator
    participant SymmetricEncryptionService
    participant JiraSprintService
    participant JIRA as JIRA REST API
    participant SprintTrackingLogRepository

    SprintTrackingOrchestrator->>SymmetricEncryptionService: decrypt(group.encryptedJiraToken)
    Note over SymmetricEncryptionService: group.encryptedJiraToken is AES-256 encrypted (NFR-7)
    SymmetricEncryptionService-->>SprintTrackingOrchestrator: plainTextToken

    SprintTrackingOrchestrator->>JiraSprintService: fetchSprintStories(group)
    Note over JiraSprintService: Authorization: Basic base64(group.jiraEmail:plainTextToken)

    JiraSprintService->>JIRA: GET /rest/agile/1.0/board?projectKeyOrId={group.jiraProjectKey}

    alt board not found or empty response
        JIRA-->>JiraSprintService: [] or 4xx
        Note over JiraSprintService: log WARN — return List.empty()
        JiraSprintService-->>SprintTrackingOrchestrator: []
    end

    Note over JiraSprintService: extract first board id = boardId

    JiraSprintService->>JIRA: GET /rest/agile/1.0/board/{boardId}/sprint?state=active

    alt no active sprint
        JIRA-->>JiraSprintService: { values: [] }
        Note over JiraSprintService: log WARN — return List.empty()
        JiraSprintService-->>SprintTrackingOrchestrator: []
    end

    Note over JiraSprintService: extract jiraSprintId from first active sprint

    JiraSprintService->>JIRA: GET /rest/agile/1.0/sprint/{jiraSprintId}/issue

    alt HTTP 200 OK
        JIRA-->>JiraSprintService: { issues: [...] }

        loop for each issue
            Note over JiraSprintService: extract issueKey, assigneeGithubUsername, storyPoints (nullable), description
        end

        JiraSprintService-->>SprintTrackingOrchestrator: List<JiraIssueDto>

        loop for each JiraIssueDto
            SprintTrackingOrchestrator->>SprintTrackingLogRepository: save(SprintTrackingLog { group, sprint, issueKey, assigneeGithubUsername, storyPoints, prMerged=null, aiPrResult=PENDING, aiDiffResult=PENDING, fetchedAt=now() })
        end

    else HTTP 401 Unauthorized
        JIRA-->>JiraSprintService: 401
        Note over JiraSprintService: log WARN — return List.empty()
        JiraSprintService-->>SprintTrackingOrchestrator: []

    else HTTP 404 Not Found
        JIRA-->>JiraSprintService: 404
        Note over JiraSprintService: log WARN — return List.empty()
        JiraSprintService-->>SprintTrackingOrchestrator: []

    else Network Error / Timeout
        JIRA-->>JiraSprintService: IOException
        Note over JiraSprintService: log error — return List.empty()
        JiraSprintService-->>SprintTrackingOrchestrator: []
    end
```
