# Sequence Diagram — P5 Sub-Process 5.0
## End-of-Sprint Tracking Orchestrator (Automated Pipeline)

> Scheduler: `@Scheduled(cron = "0 0 1 * * *")` — daily at 01:00 UTC
> Issues: #153 (Orchestrator), #148 (SprintTrackingLogRepository), #149 (JIRA), #150 (GitHub), #151 (AI PR), #152 (AI Diff)
> Pattern: self-proxy + `REQUIRES_NEW` per group — identical to `SanitizationService.java`
> No REST endpoint — triggered by cron OR by `SprintTrackingOrchestrator.triggerForSprint()` (called from 5.7)
> **MIGRATION NOTE:** `Sprint` entity requires a new `termId` column. Must be `@Column(nullable = true)` — Hibernate `ddl-auto=update` runs `ALTER TABLE sprint ADD COLUMN term_id` on an existing table; NOT NULL without a default will fail if rows exist. Set via `TermConfigService.getActiveTermId()` inside `SprintService.createSprint()`.

### runDailyTracking() — Scheduler Entry Point

```mermaid
sequenceDiagram
    actor Scheduler
    participant SprintTrackingOrchestrator
    participant SprintRepository
    participant ProjectGroupRepository
    participant Self as SprintTrackingOrchestrator (Self-Proxy)

    Scheduler->>SprintTrackingOrchestrator: runDailyTracking()
    Note over SprintTrackingOrchestrator: No @Transactional on outer method

    SprintTrackingOrchestrator->>SprintRepository: findByEndDate(yesterday)
    SprintRepository-->>SprintTrackingOrchestrator: List<Sprint>

    loop for each ended Sprint
        SprintTrackingOrchestrator->>ProjectGroupRepository: findByTermIdAndStatusIn(termId, [TOOLS_BOUND, ADVISOR_ASSIGNED])
        ProjectGroupRepository-->>SprintTrackingOrchestrator: List<ProjectGroup>

        loop for each ProjectGroup
            SprintTrackingOrchestrator->>Self: processGroup(group, sprint)
            Note over Self: @Transactional(propagation = REQUIRES_NEW)

            Self->>Self: fetchJiraStories → save SprintTrackingLog rows
            Self->>Self: matchGithubBranches → update prNumber, prMerged
            Self->>Self: validatePRReview → update aiPrResult
            Self->>Self: validateIssueDiff → update aiDiffResult

            alt any exception in processGroup()
                Note over Self: catch → log WARN → continue loop
                Note over Self: one group failing does NOT block others
            end
        end
    end
```

### processGroup() — Per-Group Transaction Detail

```mermaid
sequenceDiagram
    participant Self as SprintTrackingOrchestrator (Self-Proxy)
    participant SymmetricEncryptionService
    participant JiraSprintService
    participant GithubSprintService
    participant AiValidationService
    participant SprintTrackingLogRepository

    Note over Self: @Transactional(propagation = REQUIRES_NEW)

    Self->>SymmetricEncryptionService: decrypt(group.encryptedJiraToken)
    SymmetricEncryptionService-->>Self: plainTextJiraToken

    Self->>JiraSprintService: fetchSprintStories(group)
    JiraSprintService-->>Self: List<JiraIssueDto>

    Self->>SprintTrackingLogRepository: saveAll(rows with aiPrResult=PENDING, aiDiffResult=PENDING)

    Self->>SymmetricEncryptionService: decrypt(group.encryptedGithubPat)
    SymmetricEncryptionService-->>Self: plainTextGithubPat

    loop for each issue
        Self->>GithubSprintService: findBranchByIssueKey(group, issueKey)
        GithubSprintService-->>Self: Optional<String> branchName

        alt branch found
            Self->>GithubSprintService: findMergedPR(group, branchName)
            GithubSprintService-->>Self: Optional<GithubPrDto>
        end

        Self->>SprintTrackingLogRepository: update prNumber, prMerged

        alt prMerged = true
            Self->>GithubSprintService: fetchPRReviewComments(group, prNumber)
            GithubSprintService-->>Self: List<String> comments
            Self->>AiValidationService: validatePRReview(comments)
            AiValidationService-->>Self: AiValidationResult
            Self->>SprintTrackingLogRepository: update aiPrResult

            Self->>GithubSprintService: fetchFileDiffs(group, prNumber)
            GithubSprintService-->>Self: List<GithubFileDiffDto>
            Self->>AiValidationService: validateIssueDiff(issueDescription, diffs)
            AiValidationService-->>Self: AiValidationResult
            Self->>SprintTrackingLogRepository: update aiDiffResult

        else prMerged = false or null
            Self->>SprintTrackingLogRepository: set aiPrResult=SKIPPED, aiDiffResult=SKIPPED
        end
    end
```
