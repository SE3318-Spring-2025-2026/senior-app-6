### AiValidationService.validatePRReview(comments)

```mermaid
sequenceDiagram
    participant SprintTrackingOrchestrator
    participant GithubSprintService
    participant GitHub as GitHub REST API
    participant SystemConfigRepository
    participant SymmetricEncryptionService
    participant AiValidationService
    participant LLM as LLM API
    participant SprintTrackingLogRepository

    SprintTrackingOrchestrator->>SystemConfigRepository: findById("llm_api_key")

    alt llm_api_key found
        SystemConfigRepository-->>SprintTrackingOrchestrator: SystemConfig { value: encryptedKey }
        SprintTrackingOrchestrator->>SymmetricEncryptionService: decrypt(encryptedKey)
        SymmetricEncryptionService-->>SprintTrackingOrchestrator: plainTextLlmKey
    else llm_api_key missing
        SystemConfigRepository-->>SprintTrackingOrchestrator: Optional.empty()
        Note over SprintTrackingOrchestrator: log WARN at startup — AI results will default to WARN
    end

    loop for each SprintTrackingLog row
        alt prMerged = true
            SprintTrackingOrchestrator->>GithubSprintService: fetchPRReviewComments(group, prNumber)
            GithubSprintService->>GitHub: GET /repos/{org}/{repo}/pulls/{prNumber}/reviews
            GitHub-->>GithubSprintService: [ { body: "LGTM" }, ... ]
            GithubSprintService-->>SprintTrackingOrchestrator: List<String> commentBodies

            alt commentBodies is empty
                Note over SprintTrackingOrchestrator: no review — skip LLM call
                SprintTrackingOrchestrator->>SprintTrackingLogRepository: update aiPrResult=SKIPPED

            else commentBodies has content
                SprintTrackingOrchestrator->>AiValidationService: validatePRReview(commentBodies)
                Note over AiValidationService: Prompt (PR_REVIEW_PROMPT): "respond with PASS / WARN / FAIL"
                AiValidationService->>LLM: POST with review comments

                alt LLM responds PASS / WARN / FAIL
                    LLM-->>AiValidationService: "PASS" | "WARN" | "FAIL"
                    AiValidationService-->>SprintTrackingOrchestrator: AiValidationResult
                    SprintTrackingOrchestrator->>SprintTrackingLogRepository: update aiPrResult

                else LLM timeout (>10s)
                    LLM-->>AiValidationService: ReadTimeoutException
                    AiValidationService-->>SprintTrackingOrchestrator: WARN

                else LLM HTTP 5xx or unrecognized output
                    LLM-->>AiValidationService: error
                    AiValidationService-->>SprintTrackingOrchestrator: WARN
                end

                SprintTrackingOrchestrator->>SprintTrackingLogRepository: update aiPrResult
            end

        else prMerged = false or null
            SprintTrackingOrchestrator->>SprintTrackingLogRepository: update aiPrResult=SKIPPED
        end
    end
```
