### AiValidationService.validateIssueDiff(issueDescription, fileDiffs)

```mermaid
sequenceDiagram
    participant SprintTrackingOrchestrator
    participant GithubSprintService
    participant GitHub as GitHub REST API
    participant AiValidationService
    participant LLM as LLM API
    participant SprintTrackingLogRepository

    Note over AiValidationService: Reuses HTTP client, LLM key, and error handler from #151
    Note over AiValidationService: Only adds DIFF_VALIDATION_PROMPT — no new infrastructure

    loop for each SprintTrackingLog row
        alt prMerged = true
            SprintTrackingOrchestrator->>GithubSprintService: fetchFileDiffs(group, prNumber)
            GithubSprintService->>GitHub: GET /repos/{org}/{repo}/pulls/{prNumber}/files
            GitHub-->>GithubSprintService: [ { filename: "Foo.java", patch: "@@ ..." }, ... ]
            GithubSprintService-->>SprintTrackingOrchestrator: List<GithubFileDiffDto>

            alt fileDiffs is empty
                Note over SprintTrackingOrchestrator: no code changes — skip LLM call
                SprintTrackingOrchestrator->>SprintTrackingLogRepository: update aiDiffResult=SKIPPED

            else issueDescription is blank
                Note over SprintTrackingOrchestrator: no issue description — skip LLM call
                SprintTrackingOrchestrator->>SprintTrackingLogRepository: update aiDiffResult=SKIPPED

            else fileDiffs and issueDescription both have content
                SprintTrackingOrchestrator->>AiValidationService: validateIssueDiff(issueDescription, fileDiffs)
                Note over AiValidationService: Prompt (DIFF_VALIDATION_PROMPT): "respond with PASS / WARN / FAIL"
                Note over AiValidationService: Concatenates all patch strings + issueDescription into one request

                AiValidationService->>LLM: POST with issue description + concatenated diffs

                alt LLM responds PASS / WARN / FAIL
                    LLM-->>AiValidationService: "PASS" | "WARN" | "FAIL"
                    AiValidationService-->>SprintTrackingOrchestrator: AiValidationResult
                    SprintTrackingOrchestrator->>SprintTrackingLogRepository: update aiDiffResult

                else LLM timeout / 5xx / unrecognized output
                    LLM-->>AiValidationService: error
                    AiValidationService-->>SprintTrackingOrchestrator: WARN
                    SprintTrackingOrchestrator->>SprintTrackingLogRepository: update aiDiffResult=WARN
                end
            end

        else prMerged = false or null
            SprintTrackingOrchestrator->>SprintTrackingLogRepository: update aiDiffResult=SKIPPED
        end
    end
```
