# Sequence Diagram — P5 Sub-Process 5.2
## GitHub Branch & PR Matching

> Called by: `SprintTrackingOrchestrator.processGroup()` (see 5.0)
> Issues: #150 (GithubSprintService), #148 (SprintTrackingLog entity)
> Spec: IR-3, P5 Steps 3–4, NFR-7
> IMPORTANT: `GithubSprintService` is a completely separate class from `GithubService.java` — do NOT extend it

---

### GithubSprintService.findBranchByIssueKey() + findMergedPR()

```mermaid
sequenceDiagram
    participant SprintTrackingOrchestrator
    participant SymmetricEncryptionService
    participant GithubSprintService
    participant GitHub as GitHub REST API
    participant SprintTrackingLogRepository

    SprintTrackingOrchestrator->>SymmetricEncryptionService: decrypt(group.encryptedGithubPat)
    SymmetricEncryptionService-->>SprintTrackingOrchestrator: plainTextPat

    loop for each SprintTrackingLog row (one per JIRA issue)
        SprintTrackingOrchestrator->>GithubSprintService: findBranchByIssueKey(group, issueKey)
        Note over GithubSprintService: Authorization: Bearer {plainTextPat}

        GithubSprintService->>GitHub: GET /repos/{githubOrgName}/{repo}/branches

        alt branch name starts with issueKey
            GitHub-->>GithubSprintService: [ { name: "feature/SPM-42-fix", ... } ]
            GithubSprintService-->>SprintTrackingOrchestrator: Optional.of("feature/SPM-42-fix")

            SprintTrackingOrchestrator->>GithubSprintService: findMergedPR(group, branchName)
            GithubSprintService->>GitHub: GET /repos/{githubOrgName}/{repo}/pulls?head={branchName}&state=all

            alt state=closed AND merged_at != null
                GitHub-->>GithubSprintService: [{ number: 12, state: "closed", merged_at: "2026-05-10T..." }]
                GithubSprintService-->>SprintTrackingOrchestrator: Optional.of(GithubPrDto { prNumber=12, merged=true })
                SprintTrackingOrchestrator->>SprintTrackingLogRepository: update prNumber=12, prMerged=true

            else PR exists but merged_at = null (abandoned / open)
                GitHub-->>GithubSprintService: [{ number: 12, state: "closed", merged_at: null }]
                GithubSprintService-->>SprintTrackingOrchestrator: Optional.of(GithubPrDto { prNumber=12, merged=false })
                SprintTrackingOrchestrator->>SprintTrackingLogRepository: update prNumber=12, prMerged=false

            else no PR for this branch
                GitHub-->>GithubSprintService: []
                GithubSprintService-->>SprintTrackingOrchestrator: Optional.empty()
                SprintTrackingOrchestrator->>SprintTrackingLogRepository: update prMerged=null
            end

        else no branch matches issueKey
            GitHub-->>GithubSprintService: [] (no matching branch)
            GithubSprintService-->>SprintTrackingOrchestrator: Optional.empty()
            SprintTrackingOrchestrator->>SprintTrackingLogRepository: update prMerged=null
        end

        alt GitHub API error
            GitHub-->>GithubSprintService: IOException / 4xx / 5xx
            Note over GithubSprintService: log error — return Optional.empty()
            SprintTrackingOrchestrator->>SprintTrackingLogRepository: update prMerged=null
        end
    end
```
