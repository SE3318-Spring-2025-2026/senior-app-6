# Sequence Diagram — P5 Sub-Process 5.5b
## Advisor Scrum Grade Submission

> Endpoints: `POST /api/advisor/sprints/{sprintId}/groups/{groupId}/grade`, `GET /api/advisor/sprints/{sprintId}/groups/{groupId}/grade`
> Issues: #154 (ScrumGradingService + AdvisorSprintController)
> JWT principal = Staff UUID, role = PROFESSOR
> Spec: FR-6, P5 Steps 8–10

---

### POST /api/advisor/sprints/{sprintId}/groups/{groupId}/grade

```mermaid
sequenceDiagram
    actor Advisor
    participant AdvisorSprintController
    participant GlobalExceptionHandler
    participant ScrumGradingService
    participant ProjectGroupRepository
    participant ScrumGradeRepository

    Advisor->>AdvisorSprintController: POST /api/advisor/sprints/{sprintId}/groups/{groupId}/grade
    Note over AdvisorSprintController: Authorization: Bearer <staff_jwt> role=PROFESSOR
    Note over AdvisorSprintController: Body: { "pointA_grade": "A", "pointB_grade": "B" }

    alt invalid grade value (e.g. "G")
        Note over AdvisorSprintController: Jackson fails to deserialize ScrumGradeValue enum
        AdvisorSprintController->>GlobalExceptionHandler: HttpMessageNotReadableException
        GlobalExceptionHandler-->>Advisor: 400 {"message": "pointA_grade must be one of: A, B, C, D, F"}
    end

    AdvisorSprintController->>AdvisorSprintController: extract advisorId from SecurityContext
    AdvisorSprintController->>ScrumGradingService: submitGrade(advisorId, groupId, sprintId, pointAGrade, pointBGrade)

    ScrumGradingService->>ProjectGroupRepository: findById(groupId)
    ProjectGroupRepository-->>ScrumGradingService: Optional<ProjectGroup>

    alt group not found
        ScrumGradingService-->>AdvisorSprintController: throw GroupNotFoundException
        AdvisorSprintController-->>Advisor: 404 {"message": "Group not found"}
    end

    Note over ScrumGradingService: Ownership check: group.getAdvisor() == null OR group.getAdvisor().getId() != advisorId
    Note over ScrumGradingService: Use group.getAdvisor() — there is no group.getAdvisorId() field

    alt ownership check fails
        ScrumGradingService-->>AdvisorSprintController: throw ForbiddenException
        AdvisorSprintController-->>Advisor: 403 {"message": "You are not the advisor for this group"}
    end

    ScrumGradingService->>ScrumGradeRepository: existsByGroupIdAndSprintId(groupId, sprintId)

    alt no existing grade — first submission
        ScrumGradeRepository-->>ScrumGradingService: false
        Note over ScrumGradingService: create ScrumGrade: gradedAt=now(), updatedAt=null
        ScrumGradingService->>ScrumGradeRepository: save(newScrumGrade)
        ScrumGradeRepository-->>ScrumGradingService: saved ScrumGrade
        ScrumGradingService-->>AdvisorSprintController: ScrumGradeResponse
        AdvisorSprintController-->>Advisor: 201 { gradeId, groupId, sprintId, pointA_grade, pointB_grade, advisorId, gradedAt }

    else grade exists — update
        ScrumGradeRepository-->>ScrumGradingService: true
        ScrumGradingService->>ScrumGradeRepository: findByGroupIdAndSprintId(groupId, sprintId)
        ScrumGradeRepository-->>ScrumGradingService: ScrumGrade
        Note over ScrumGradingService: update pointAGrade, pointBGrade, updatedAt=now() — gradedAt unchanged
        ScrumGradingService->>ScrumGradeRepository: save(existing)
        ScrumGradeRepository-->>ScrumGradingService: updated ScrumGrade
        ScrumGradingService-->>AdvisorSprintController: ScrumGradeResponse
        AdvisorSprintController-->>Advisor: 200 { gradeId, groupId, sprintId, pointA_grade, pointB_grade, advisorId, gradedAt, updatedAt }
    end
```

### GET /api/advisor/sprints/{sprintId}/groups/{groupId}/grade

```mermaid
sequenceDiagram
    actor Advisor
    participant AdvisorSprintController
    participant ScrumGradingService
    participant ProjectGroupRepository
    participant ScrumGradeRepository

    Advisor->>AdvisorSprintController: GET /api/advisor/sprints/{sprintId}/groups/{groupId}/grade
    Note over AdvisorSprintController: Authorization: Bearer <staff_jwt> role=PROFESSOR
    Note over AdvisorSprintController: Frontend calls this immediately after POST to refresh live panel (FR-6)

    AdvisorSprintController->>AdvisorSprintController: extract advisorId from SecurityContext
    AdvisorSprintController->>ScrumGradingService: getGrade(advisorId, groupId, sprintId)

    ScrumGradingService->>ProjectGroupRepository: findById(groupId)
    ProjectGroupRepository-->>ScrumGradingService: Optional<ProjectGroup>

    alt group not found
        ScrumGradingService-->>AdvisorSprintController: throw GroupNotFoundException
        AdvisorSprintController-->>Advisor: 404 {"message": "Group not found"}
    end

    Note over ScrumGradingService: Ownership check: group.getAdvisor() == null OR group.getAdvisor().getId() != advisorId

    alt ownership check fails
        ScrumGradingService-->>AdvisorSprintController: throw ForbiddenException
        AdvisorSprintController-->>Advisor: 403 {"message": "You are not the advisor for this group"}
    end

    ScrumGradingService->>ScrumGradeRepository: findByGroupIdAndSprintId(groupId, sprintId)

    alt grade found
        ScrumGradeRepository-->>ScrumGradingService: Optional.of(ScrumGrade)
        ScrumGradingService-->>AdvisorSprintController: ScrumGradeResponse
        AdvisorSprintController-->>Advisor: 200 { gradeId, pointA_grade, pointB_grade, gradedAt, updatedAt }

    else no grade submitted yet
        ScrumGradeRepository-->>ScrumGradingService: Optional.empty()
        ScrumGradingService-->>AdvisorSprintController: throw NotFoundException
        AdvisorSprintController-->>Advisor: 404 {"message": "No grade submitted yet for this group and sprint"}
    end
```
