### POST /api/coordinator/sprints/{sprintId}/refresh

```mermaid
sequenceDiagram
    actor Coordinator
    participant CoordinatorController
    participant SprintTrackingOrchestrator
    participant SprintRepository
    participant SprintTrackingLogRepository

    Coordinator->>CoordinatorController: POST /api/coordinator/sprints/{sprintId}/refresh
    Note over CoordinatorController: Authorization: Bearer <staff_jwt> role=COORDINATOR
    Note over CoordinatorController: Optional query param: ?force=true

    CoordinatorController->>SprintRepository: findById(sprintId)
    SprintRepository-->>CoordinatorController: Optional<Sprint>

    alt sprint not found
        CoordinatorController-->>Coordinator: 404 {"message": "Sprint not found"}
    end

    CoordinatorController->>SprintTrackingOrchestrator: triggerForSprint(sprintId, force)

    alt force=false AND sprint.endDate >= today
        SprintTrackingOrchestrator-->>CoordinatorController: throw BusinessRuleException
        CoordinatorController-->>Coordinator: 400 {"message": "Sprint has not ended yet — use ?force=true to refresh early"}
    end

    Note over SprintTrackingOrchestrator: Idempotent reset — delete all existing rows before re-fetching
    SprintTrackingOrchestrator->>SprintTrackingLogRepository: deleteBySprintId(sprintId)

    Note over SprintTrackingOrchestrator: Run full pipeline synchronously — same self-proxy REQUIRES_NEW flow as daily run
    SprintTrackingOrchestrator-->>CoordinatorController: SprintRefreshStats { groupsProcessed, issuesFetched, aiValidationsRun }

    CoordinatorController-->>Coordinator: 200 { sprintId, groupsProcessed, issuesFetched, aiValidationsRun, triggeredAt }
```

### GET /api/coordinator/sprints/{sprintId}/overview

```mermaid
sequenceDiagram
    actor Coordinator
    participant CoordinatorController
    participant SprintService
    participant SprintRepository
    participant ProjectGroupRepository
    participant SprintTrackingLogRepository
    participant ScrumGradeRepository

    Coordinator->>CoordinatorController: GET /api/coordinator/sprints/{sprintId}/overview
    Note over CoordinatorController: Authorization: Bearer <staff_jwt> role=COORDINATOR

    CoordinatorController->>SprintRepository: findById(sprintId)
    SprintRepository-->>CoordinatorController: Optional<Sprint>

    alt sprint not found
        CoordinatorController-->>Coordinator: 404 {"message": "Sprint not found"}
    end

    CoordinatorController->>SprintService: getSprintOverview(sprintId)

    Note over SprintService: termId resolved server-side — never from client
    SprintService->>ProjectGroupRepository: findByTermIdAndStatusIn(termId, [TOOLS_BOUND, ADVISOR_ASSIGNED])
    Note over ProjectGroupRepository: ALL groups — not filtered to any advisor
    ProjectGroupRepository-->>SprintService: List<ProjectGroup>

    loop for each group
        SprintService->>SprintTrackingLogRepository: findByGroupIdAndSprintId(group.id, sprintId)
        SprintTrackingLogRepository-->>SprintService: List<SprintTrackingLog>

        SprintService->>ScrumGradeRepository: findByGroupIdAndSprintId(group.id, sprintId)
        ScrumGradeRepository-->>SprintService: Optional<ScrumGrade>

        Note over SprintService: advisorName = group.getAdvisor()?.getName() — null for TOOLS_BOUND groups
        Note over SprintService: gradeSubmitted = grade.isPresent() — pointA_grade/pointB_grade null if absent
    end

    SprintService-->>CoordinatorController: List<SprintGroupOverview>
    CoordinatorController-->>Coordinator: 200 { sprintId, groups: [{ groupId, groupName, advisorName, totalIssues, mergedPRs, aiCounts, gradeSubmitted, pointA_grade, pointB_grade }, ...] }
```
